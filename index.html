API for Knowledge Platforms (API4KP)

Technical Specification and Implementation

The API4KP standard is based on a model-driven architecture that combines UML models, OWL ontologies and OpenAPI specifcations to define a platform, technology and vendor independent way to build Knowledge Based Systems.
