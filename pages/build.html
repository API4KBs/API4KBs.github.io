The codebase consists of a multi-module project with dependencies.
It is recommended to build the modules in the given order.

Requirements:

  • JDK 8
    (JDK 11+ is not yet supported by some of the sub-dependencies)
  • Maven 3.3.9+
    (Tested with Maven 3.3.9, 3.5.x and 3.6.x)

Build Instructions

  1. Checkout and Build the modules in order, following/adpating this script
