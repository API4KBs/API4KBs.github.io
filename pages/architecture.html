The codebase consists of a multi-module project with dependencies. It is recommended to build the modules in the given order.' Scripts are provided as a reference

Requirements:

  • JDK 8
  • Maven 3.3.9+

Notice that JDK 11+ is not yet supported by some of the sub-dependencies)

  1. Clone / install the configurable archetype plugin
    1. git clone https://github.com/API4KBs/maven-cfg-archetype-plugin.git
    2. cd maven-cfg-archetype-plugin
    3. (checkout develop) 
    4. mvn clean install
  2. Clone / install the API4KP specifications
    1. git clone https://github.com/API4KBs/api4kbs.git
    2. cd api4kbs
    3. (checkout develop) 
    4. mvn clean install
  3. Clone / install the master BOM 
    1. git clone https://github.com/API4KBs/kmdp-bom.git
    2. cd kmdp-bom
    3. (checkout develop) 
    4. mvn clean install
  4. Clone / install the core libraries
    1. git clone https://github.com/API4KBs/kmdp-models.git
    2. cd kmdp-models
    3. (checkout develop) 
    4. mvn clean install
  5. Clone / install the standard models
    1. git clone https://github.com/API4KBs/kmdp-standard-models-impl.git
    2. cd kmdp-standard-models-impl
    3. (checkout develop) 
    4. mvn clean install
  6. Clone / install the OpenAPI/Spring Archetype
    1. git clone https://github.com/API4KBs/kmdp-archetype.git
    2. cd kmdp-archetype
    3. (checkout develop) 
    4. mvn clean install
  7. Clone / install the API framework
    1. git clone https://github.com/API4KBs/kmdp-api.git
    2. cd kmdp-api
    3. (checkout develop) 
    4. mvn clean package -Prefresh
    5. mvn clean install
  8. Clone / install the Knowledge Artifact Repository implementation
    1. git clone https://github.com/API4KBs/kmdp-artifact-repository.git
    2. cd kmdp-artifact-repository
    3. (checkout develop) 
    4. mvn clean install
  9. Clone / install the Language / Transrepresentation implementation
    1. git clone https://github.com/API4KBs/kmdp-language.git
    2. cd kmdp-language
    3. (checkout develop) 
    4. mvn clean install
  10. Clone / install the Knowledge Asset Repository implementation
    1. git clone https://github.com/API4KBs/kmdp-asset-repository.git
    2. cd kmdp-asset-repository
    3. (checkout develop) 
    4. mvn clean install
